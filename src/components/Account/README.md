# Account
Show user account resume

## Usage

```js  
const Example = () => (
  <Account
    ammount={10}
    onBottomPress={() => {}}
  />);
```

### Properties

| Prop                    | Description                      | Required      | Default | Format | 
| ----------------------- | ---------------------------------| ------------- | ------- | ------ |
| **`ammount`**           | Ammount of user in account       | true          | 0       | number |
| **`onBottomPress`**     | Action when user click bottom of card      | true          | () => {}       | function |
